# NestJS - TODO

Simple REST api for manage todos

## Installation

```bash
$ yarn

# Generate jwt secret key
$ yarn jwt:secret
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
